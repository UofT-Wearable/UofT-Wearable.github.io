Project for 2024 - 2025

We are a new EngSoc Trial Status club, and our mandate is to develop wearable expertise in the community through three areas:

  • Embedded electronics
  • Form and function
  • Companion software applications

Our first project is a Wearable ECG (Electrocardiogram) sensor, which measures the electrical signals of the heart. The scope of this project includes:

  • Designing and placing physical electrodes
  • Developing hardware connections
  • Establishing hardware communication to sensors on a microcontroller
  • Implementing wireless Bluetooth communication from the microcontroller to a smartphone
  • Building a smartphone application

Upon successful prototyping, we plan to move into PCB design and manufacturing to create a prototype with a realistic, enclosed electronic form factor.

This project benefits the Skule community by creating a space to learn and practice wearable technology development. We will host teaching sessions on topics such as:

  • Programming embedded systems using the Zephyr RTOS
  • Hardware-sensor communication
  • Application development using Flutter

Our biomedical focus explores the development of an all-day wearable ECG monitoring solution. If successful, this project could significantly impact healthcare.

We are using the MAX30001G Evaluation Kit, which allows us to prototype various 2-electrode ECG devices. This industry-standard sensor provides invaluable experience with biomedical sensors used worldwide in wearable heart monitors. Developing a full-fledged wearable prototype with a PCB is essential, and this sensor chip is critical for success.
